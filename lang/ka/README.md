 рЃЮрЃарЃўрЃњрЃўрЃюрЃљрЃџрЃўрЃА рЃћрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃў [README] (https://github.com/cobbzilla/yuebing/blob/master/README.md)

 рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџ GitHub рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃА рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ, рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ, рЃњрЃљрЃљрЃЎрЃћрЃЌрЃЮрЃЌ:

 * рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ рЃерЃћрЃЏрЃЮрЃњрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЌ рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃўрЃА рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃљ

 рЃњрЃБрЃџрЃгрЃарЃцрЃћрЃџрЃљрЃЊ рЃЋрЃљрЃцрЃљрЃАрЃћрЃЉ рЃЌрЃБрЃюрЃЊрЃљрЃф рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃфрЃўрЃарЃћ [рЃЌрЃЋрЃўрЃБрЃарЃў рЃерЃћрЃюрЃљрЃбрЃљрЃюрЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ Patreon] (https://www.patreon.com/cobbzilla)

 рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃ«рЃљрЃќрЃўрЃА рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ `npm` рЃљрЃю `yarn` :

    npm install -g hokeylization
    yarn global add hokeylization

 рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ `lite` рЃЋрЃћрЃарЃАрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃфрЃўрЃџрЃћрЃЉрЃўрЃЌ рЃЏрЃфрЃўрЃарЃћрЃљ:

    npm install -g hokeylization-lite
    yarn global add hokeylization-lite

 рЃерЃћрЃЏрЃЊрЃћрЃњ рЃерЃћрЃ«рЃћрЃЊрЃћрЃЌ рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃА `hokey` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА:

    hokey --help
    hokey -h

 рЃњрЃАрЃБрЃарЃЌ рЃюрЃљрЃ«рЃЮрЃЌ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃћрЃюрЃљрЃќрЃћ рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃћрЃюрЃљрЃќрЃћ?

 `hokey` рЃфрЃЊрЃўрЃџрЃЮрЃЉрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃЏрЃЮрЃўрЃфрЃюрЃЮрЃА рЃћрЃюрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃњрЃљрЃарЃАрЃўрЃА рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃЊрЃљрЃю

 рЃЌрЃЦрЃЋрЃћрЃю рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃљрЃўрЃФрЃБрЃџрЃЮрЃЌ рЃћрЃюрЃљ `LC_ALL` рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ:

    LC_ALL=it hokey --help

 рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃю рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ `hokeylization-lite` , рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃа рЃћрЃюрЃљрЃќрЃћ

 ## р▓љрЃгрЃДрЃЮрЃЉрЃљ
 рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `GOOGLE_TRANSLATE_PROJECT_ID` рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃў Google Translate рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

 рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ `GOOGLE_APPLICATION_CREDENTIALS` рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў JSON рЃАрЃћрЃарЃЌрЃўрЃцрЃўрЃЎрЃљрЃбрЃћрЃЉрЃќрЃћ, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ
 рЃўрЃЏрЃўрЃА рЃњрЃљрЃарЃЎрЃЋрЃћрЃЋрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ Google рЃдрЃарЃБрЃЉрЃћрЃџрЃќрЃћ (рЃћрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃАрЃљрЃ«рЃљрЃџрЃўрЃАрЃЮ)

 рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃю рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃгрЃДрЃљрЃарЃЮрЃА рЃЎрЃЮрЃЊрЃўрЃЊрЃљрЃю, рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃюрЃљрЃЌрЃљрЃЋрЃАрЃЮрЃЌ рЃўрЃАрЃўрЃюрЃў `.env` рЃцрЃљрЃўрЃџрЃерЃў рЃгрЃДрЃљрЃарЃЮрЃерЃў
 рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ, рЃўрЃАрЃўрЃюрЃў рЃЕрЃљрЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА [dotenv]-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ (https://www.npmjs.com/package/dotenv)

 ## JavaScript рЃАрЃўрЃЏрЃћрЃЉрЃўрЃљрЃюрЃў рЃарЃћрЃАрЃБрЃарЃАрЃўрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃЌрЃљрЃарЃњрЃЏрЃюрЃљ
 рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃўрЃЏрЃћрЃЉрЃўрЃљрЃюрЃў рЃфрЃ«рЃарЃўрЃџрЃў **рЃБрЃюрЃЊрЃљ** рЃўрЃДрЃЮрЃА JavaScript рЃцрЃљрЃўрЃџрЃерЃў рЃљрЃЏ рЃЮрЃарЃўрЃЊрЃљрЃю рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃцрЃЮрЃарЃЏрЃўрЃЌ:

 ES6 рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў:

    export default {
      string_key: "some value",
      another_key: "another value",
      ... more keys ...
    }

 CommonJS рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў

    module.exports = {
      string_key: "some value",
      another_key: "another value",
      ... more keys ...
    }

 рЃЌрЃБ рЃљрЃЏ рЃцрЃљрЃўрЃџрЃА рЃћрЃарЃЦрЃЋрЃљ `myfile.en.js` , рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЌрЃљрЃарЃњрЃЏрЃюрЃЮрЃЌ рЃћрЃАрЃърЃљрЃюрЃБрЃа рЃЊрЃљ рЃњрЃћрЃарЃЏрЃљрЃюрЃБрЃџ рЃћрЃюрЃћрЃЉрЃќрЃћ:

    hokey -l es,de -o myfile.LANG.js myfile.en.js

 рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў `LANG` рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃБрЃџрЃўрЃљ -- рЃћрЃА рЃљрЃарЃўрЃА рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃљ рЃљрЃЏ рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃерЃў!

 `LANG` рЃЕрЃљрЃюрЃљрЃфрЃЋрЃџрЃћрЃЉрЃБрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃћрЃюрЃўрЃА рЃЎрЃЮрЃЊрЃўрЃЌ

 рЃљрЃЏрЃарЃўрЃњрЃљрЃЊ, рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃЦрЃЏрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА:

    myfile.es.js
    myfile.de.js

 `-l` / `--languages` рЃљрЃарЃўрЃА ISO рЃћрЃюрЃўрЃА рЃЎрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃЏрЃФрЃўрЃЏрЃўрЃЌ рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃў рЃАрЃўрЃљ.
 [рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃЌ Google Translate](https://cloud.google.com/translate/docs/languages)

 рЃЌрЃБ рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃў рЃБрЃЎрЃЋрЃћ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА, рЃўрЃА рЃњрЃљрЃюрЃўрЃ«рЃўрЃџрЃћрЃЉрЃљ, рЃарЃљрЃЌрЃљ рЃЊрЃљрЃЊрЃњрЃўрЃюрЃЊрЃћрЃА рЃарЃЮрЃЏрЃћрЃџрЃў рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃў рЃБрЃЎрЃЋрЃћ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА.
 рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃў рЃљрЃа рЃўрЃЌрЃљрЃарЃњрЃЏрЃюрЃћрЃЉрЃљ. рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃћрЃЉрЃў рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЊрЃљ рЃЊрЃљрЃарЃЌрЃБрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ
 JS рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃЉрЃЮрЃџрЃЮрЃЏрЃЊрЃћ. рЃЏрЃЌрЃћрЃџрЃў рЃцрЃљрЃўрЃџрЃў рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃўрЃгрЃћрЃарЃћрЃЉрЃљ.

 рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃџрЃљрЃЋрЃўрЃерЃўрЃА рЃ«рЃћрЃџрЃљрЃ«рЃљрЃџрЃў рЃЌрЃљрЃарЃњрЃЏрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `-f` / `--force` рЃЮрЃцрЃфрЃўрЃљ

 ## рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃА рЃЌрЃљрЃарЃњрЃЏрЃюрЃљ
 рЃЌрЃЦрЃЋрЃћрЃю рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЌрЃљрЃарЃњрЃЏрЃюрЃЮрЃЌ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ. рЃ░рЃЮрЃЎрЃћрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃљрЃЊ рЃћрЃгрЃЋрЃћрЃЋрЃљ рЃДрЃЋрЃћрЃџрЃљ
 рЃерЃћрЃўрЃюрЃљрЃ«рЃћрЃЌ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃЏрЃўрЃАрЃў рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў Google Translate-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃЊрЃљ рЃерЃћрЃўрЃюрЃљрЃ«рЃћрЃЌ рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў
 рЃўрЃЊрЃћрЃюрЃбрЃБрЃа рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃБрЃџ рЃцрЃљрЃўрЃџрЃА рЃфрЃљрЃџрЃЎрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў

 рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ рЃљрЃарЃўрЃА, рЃћрЃА рЃарЃћрЃЪрЃўрЃЏрЃў рЃЕрЃљрЃарЃЌрЃБрЃџрЃўрЃљ

 `-o` / `--outfile` рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ

 **рЃЊрЃўрЃЊрЃў рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃљ**: рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃњрЃЏрЃюрЃўрЃАрЃљрЃА **рЃљрЃа** рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃЮрЃЌ рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ
 рЃћрЃА рЃљрЃарЃўрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў! рЃЌрЃБ рЃљрЃЏрЃљрЃА рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃЌ, рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃЌ:
 * рЃБрЃАрЃљрЃАрЃарЃБрЃџрЃЮ рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃгрЃЋрЃћрЃЋрЃљ
 * рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў Google рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў
 * рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЊрЃўрЃАрЃЎрЃў
 * рЃюрЃљрЃЎрЃџрЃћрЃЉрЃљрЃЊ рЃњрЃљрЃћрЃарЃЌрЃћрЃЌ

 рЃљрЃЦ рЃљрЃарЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃўрЃЏрЃўрЃАрЃљ, рЃарЃљрЃф *рЃљрЃа рЃБрЃюрЃЊрЃљ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃЮрЃА*:

    hokey -l es -o templates/es templates # <--- DON'T DO THIS!

 рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃћрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ, рЃЌрЃљрЃарЃњрЃЏрЃюрЃўрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃўрЃгрЃћрЃарЃћрЃЉрЃљ `templates/es` рЃЊрЃљ рЃљрЃЏрЃўрЃЌ рЃ«рЃЊрЃћрЃЉрЃљ рЃљрЃ«рЃљрЃџрЃў
 рЃАрЃљрЃЌрЃљрЃарЃњрЃЏрЃюрЃў рЃгрЃДрЃљрЃарЃЮ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў, рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃАрЃўрЃюрЃў рЃљрЃарЃўрЃљрЃю `templates/` -- рЃћрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃЉрЃљ
 рЃАрЃљрЃЏрЃБрЃЊрЃљрЃЏрЃЮрЃЊ, рЃюрЃБ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉ рЃљрЃЏрЃљрЃА!

 #### рЃАрЃгрЃЮрЃарЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
 рЃЎрЃљрЃарЃњрЃў, рЃЋрЃЌрЃЦрЃЋрЃљрЃЌ, рЃарЃЮрЃЏ рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃљрЃЦрЃЋрЃЌ рЃћрЃџ.рЃцрЃЮрЃАрЃбрЃўрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃерЃљрЃЉрЃџрЃЮрЃюрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў:

    templates/email/en/welcome.txt
    templates/email/en/welcome.html
    templates/email/en/verify-account.txt
    templates/email/en/verify-account.html
    templates/email/en/reset-password.txt
    templates/email/en/reset-password.html

 рЃћрЃАрЃърЃљрЃюрЃБрЃарЃљрЃЊ рЃЊрЃљ рЃњрЃћрЃарЃЏрЃљрЃюрЃБрЃџрЃљрЃЊ рЃарЃЮрЃЏ рЃЌрЃљрЃарЃњрЃЏрЃюрЃЮрЃЌ рЃћрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў, рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ:

    hokey -l es,de -o templates/email/LANG templates/email/en

 рЃќрЃћрЃЏрЃЮрЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџрЃерЃў, `LANG` рЃљрЃарЃўрЃА рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃљ рЃЊрЃљ рЃерЃћрЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ ISO рЃћрЃюрЃўрЃА рЃЎрЃЮрЃЊрЃўрЃЌ

 рЃарЃљ рЃ«рЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃќрЃћрЃЏрЃЮрЃЌ рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў:
 * рЃерЃћрЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃў `templates/email/es` рЃЊрЃљ `templates/email/de` (рЃЌрЃБ рЃўрЃАрЃўрЃюрЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА)
 * рЃДрЃЋрЃћрЃџрЃљ рЃцрЃљрЃўрЃџрЃў `templates/email/en` рЃўрЃЌрЃљрЃарЃњрЃЏрЃюрЃћрЃЉрЃљ рЃћрЃАрЃърЃљрЃюрЃБрЃа рЃЊрЃљ рЃњрЃћрЃарЃЏрЃљрЃюрЃБрЃџ рЃћрЃюрЃћрЃЉрЃќрЃћ
 * рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃљрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃарЃћрЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў, рЃЌрЃБ рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ `-f` / `--force`
 * рЃЌрЃЦрЃЋрЃћрЃю рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ рЃўрЃЊрЃћрЃюрЃбрЃБрЃарЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА рЃЊрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃА `es` рЃЊрЃљ `de` , рЃарЃЮрЃњрЃЮрЃарЃўрЃф рЃњрЃљрЃЦрЃЋрЃЌ `en`

 ## рЃАрЃ«рЃЋрЃљ рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃћрЃЉрЃў

 ### рЃЏрЃерЃарЃљрЃџрЃў рЃАрЃўрЃарЃЉрЃўрЃџрЃў
 рЃњрЃљрЃўрЃљрЃарЃћрЃЌ `-n` / `--dry-run` рЃАрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ, рЃЌрЃБ рЃарЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЎрЃћрЃЌрЃЊрЃћрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ рЃљрЃа рЃњрЃљрЃюрЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃЮрЃЌ API рЃќрЃљрЃарЃћрЃЉрЃў рЃЊрЃљ рЃљрЃа рЃЊрЃљрЃгрЃћрЃарЃЮрЃЌ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў

 ### рЃФрЃљрЃџрЃљ
 `-f` / `--force` рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃарЃћрЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЌрЃБрЃюрЃЊрЃљрЃф рЃўрЃАрЃўрЃюрЃў рЃБрЃЎрЃЋрЃћ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃЊрЃюрЃћрЃю

 ### рЃЏрЃљрЃбрЃЕрЃў
 рЃњрЃљрЃўрЃљрЃарЃћрЃЌ `-m` / `--match` , рЃарЃљрЃЌрЃљ рЃерЃћрЃќрЃдрЃБрЃЊрЃЮрЃЌ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ-рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА

 рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃа рЃњрЃўрЃюрЃЊрЃЮрЃЊрЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃгрЃДрЃљрЃарЃЮрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў *рЃДрЃЋрЃћрЃџрЃљ* рЃцрЃљрЃўрЃџрЃўрЃА рЃЌрЃљрЃарЃњрЃЏрЃюрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃАрЃљрЃЏрЃўрЃќрЃюрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў

 `-m` / `--match` рЃЮрЃцрЃфрЃўрЃЮрЃюрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃљрЃарЃўрЃА рЃарЃћрЃњрЃћрЃЦрЃАрЃў (рЃБрЃцрЃарЃЌрЃ«рЃўрЃџрЃЊрЃўрЃЌ рЃГрЃБрЃарЃЋрЃўрЃА рЃфрЃўрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃА!), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА
 рЃарЃЮрЃЏрЃћрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃБрЃюрЃЊрЃљ рЃўрЃЌрЃљрЃарЃњрЃЏрЃюрЃЮрЃА

 рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃћрЃГрЃЋрЃў рЃњрЃћрЃърЃљрЃарЃћрЃЉрЃљрЃЌ, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЊрЃљрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃЮрЃЌ рЃћрЃА рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў `-n` / `--dry-run` , рЃарЃЮрЃЏ рЃюрЃљрЃ«рЃЮрЃЌ рЃарЃЮрЃЏрЃћрЃџрЃў рЃцрЃљрЃўрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЌрЃљрЃарЃњрЃЏрЃюрЃўрЃџрЃў

 ### рЃњрЃљрЃЏрЃЮрЃарЃўрЃфрЃ«рЃљрЃЋрЃА
 рЃќрЃЮрЃњрЃ»рЃћрЃа рЃЌрЃЦрЃЋрЃћрЃюрЃў `-m` рЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃЉрЃћрЃЋрЃа рЃцрЃљрЃўрЃџрЃА. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `-e` / `--excludes` рЃЮрЃцрЃфрЃўрЃљ рЃфрЃљрЃџрЃАрЃљрЃ«рЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃарЃўрЃфрЃ«рЃљрЃЊ
 рЃцрЃљрЃўрЃџрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃАрЃ«рЃЋрЃљрЃњрЃЋрЃљрЃарЃљрЃЊ рЃЊрЃљрЃћрЃЏрЃЌрЃ«рЃЋрЃљ

 рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЕрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃЮрЃЌ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃарЃћрЃњрЃћрЃЦрЃАрЃў, рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃўрЃЌ

 рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃўрЃЦрЃюрЃћрЃЉрЃљ: `--excludes node_modules dist \.git build tmp`

 ### рЃАрЃљрЃ«рЃћрЃџрЃБрЃарЃў
 рЃАрЃљрЃЌрЃљрЃарЃњрЃЏрЃюрЃў рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА `{{ handlebars }}` рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃА, рЃЮрЃарЃў рЃљрЃю рЃАрЃљрЃЏрЃў рЃ«рЃЋрЃћрЃБрЃџрЃў рЃЉрЃарЃћрЃЎрЃћрЃбрЃўрЃЌ

 рЃЌрЃЦрЃЋрЃћрЃю рЃљрЃџрЃЉрЃљрЃЌ *рЃљрЃа* рЃњрЃўрЃюрЃЊрЃљрЃЌ, рЃарЃЮрЃЏ рЃљрЃЏ рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЏрЃљрЃАрЃљрЃџрЃљ рЃўрЃЌрЃљрЃарЃњрЃЏрЃюрЃЮрЃА

 рЃњрЃљрЃўрЃљрЃарЃћрЃЌ `-H` / `--handlebars` рЃЊрЃарЃЮрЃерЃљ, рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў `{{ ... }}` -рЃўрЃА рЃцрЃљрЃарЃњрЃџрЃћрЃЉрЃерЃў рЃљрЃа рЃўрЃЌрЃљрЃарЃњрЃЏрЃюрЃћрЃЉрЃљ

 ### рЃЏрЃљрЃарЃЎрЃЊрЃљрЃБрЃюрЃў
 Markdown рЃљрЃарЃф рЃбрЃћрЃЦрЃАрЃбрЃўрЃљ рЃЊрЃљ рЃљрЃарЃф html, рЃљрЃЏрЃўрЃбрЃЮрЃЏ Google Translate-рЃА рЃљрЃЦрЃЋрЃА рЃњрЃљрЃарЃЎрЃЋрЃћрЃБрЃџрЃў рЃАрЃўрЃарЃЌрЃБрЃџрЃћрЃћрЃЉрЃў

 `-M` / `--markdown` рЃЊрЃарЃЮрЃерЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃА рЃ«рЃЊрЃўрЃА рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃљрЃЊ рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃА рЃЏрЃљрЃарЃЎрЃўрЃарЃћрЃЉрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

 рЃЏрЃљрЃарЃЎрЃЊрЃљрЃБрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ, рЃЌрЃБ рЃљрЃа рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ `-M` рЃЊрЃарЃЮрЃерЃљрЃА, рЃљрЃџрЃЉрЃљрЃЌ рЃўрЃърЃЮрЃЋрЃўрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃА:
 * рЃњрЃљрЃбрЃћрЃ«рЃўрЃџрЃў рЃЉрЃЏрЃБрЃџрЃћрЃЉрЃў. рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃерЃў рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃўрЃА рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮ рЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃЏрЃљрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃарЃљрЃф рЃЉрЃЏрЃБрЃџрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃЮрЃЉрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ ( `]` ), рЃЏрЃљрЃњрЃарЃљрЃЏ
 рЃАрЃљрЃюрЃљрЃЏ рЃЏрЃўрЃАрЃў рЃАрЃљрЃЏрЃўрЃќрЃюрЃћ рЃЉрЃЏрЃБрЃџрЃў рЃЊрЃљрЃўрЃгрЃДрЃћрЃЉрЃљ ( `(` )-рЃўрЃЌ).
 рЃўрЃерЃџрЃћрЃЉрЃљ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃюрЃљрЃ«рЃЋрЃўрЃАрЃљрЃА.
 * рЃЎрЃЮрЃЊрЃўрЃА рЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃў рЃўрЃЌрЃљрЃарЃњрЃЏрЃюрЃћрЃЉрЃљ. Google translate-рЃЏрЃљ рЃљрЃа рЃўрЃфрЃўрЃА, рЃарЃљрЃА рЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃА рЃЏрЃљрЃарЃЎрЃЊрЃљрЃБрЃюрЃў рЃЎрЃЮрЃЊрЃў рЃЊрЃљ рЃарЃљрЃА рЃљрЃарЃљ
 * рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃў рЃЕрЃљрЃдрЃарЃЏрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃерЃў рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃўрЃА рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃљ рЃарЃЌрЃБрЃџрЃўрЃљ
 * `backticks` "-рЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃюрЃўрЃЋрЃЌрЃћрЃЉрЃў рЃўрЃЌрЃљрЃарЃњрЃЏрЃюрЃћрЃЉрЃљ, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЌрЃўрЃЌрЃЦрЃЏрЃўрЃА рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃњрЃўрЃюрЃЊрЃљрЃЌ, рЃарЃЮрЃЏ рЃўрЃАрЃўрЃюрЃў рЃўрЃДрЃЮрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў

 рЃарЃЮрЃЊрЃћрЃАрЃљрЃф `-M` / `--markdown` рЃЊрЃарЃЮрЃерЃљ рЃЕрЃљрЃарЃЌрЃБрЃџрЃўрЃљ:
 * рЃерЃљрЃЉрЃџрЃЮрЃюрЃў `] (` рЃЎрЃЮрЃюрЃЊрЃћрЃюрЃАрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ `](` рЃљрЃЏрЃўрЃЌ рЃЊрЃљрЃцрЃўрЃЦрЃАрЃўрЃарЃЊрЃћрЃЉрЃљ рЃњрЃљрЃбрЃћрЃ«рЃўрЃџрЃў рЃЏрЃљрЃарЃЎрЃўрЃарЃћрЃЉрЃўрЃА рЃЉрЃЏрЃБрЃџрЃћрЃЉрЃў
 * рЃЕрЃљрЃдрЃарЃЏрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃўрЃА рЃўрЃарЃњрЃЋрЃџрЃўрЃЋ рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃЊрЃћрЃЉрЃљ рЃерЃћрЃцрЃБрЃЌрЃЋрЃљ "рЃљрЃа рЃљрЃарЃўрЃА рЃЌрЃљрЃарЃњрЃЏрЃюрЃўрЃџрЃў", рЃерЃћрЃўрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃА рЃАрЃљрЃЌрЃљрЃюрЃљрЃЊрЃЮ рЃЕрЃљрЃдрЃарЃЏрЃљрЃЋрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЏрЃљрЃЌ рЃљрЃа рЃЌрЃљрЃарЃњрЃЏрЃюрЃљрЃА
 * рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃўрЃарЃњрЃЋрЃџрЃўрЃЋ рЃЊрЃљрЃўрЃЊрЃћрЃЉрЃљ рЃерЃћрЃцрЃБрЃЌрЃЋрЃљ "рЃљрЃа рЃљрЃарЃўрЃА рЃЌрЃљрЃарЃњрЃЏрЃюрЃўрЃџрЃў" `backticks` " рЃцрЃљрЃарЃњрЃџрЃћрЃЉрЃерЃў, рЃарЃљрЃЌрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃўрЃџрЃў рЃўрЃДрЃЮрЃА, рЃарЃЮрЃЏ рЃўрЃАрЃўрЃюрЃў рЃљрЃа рЃўрЃЌрЃљрЃарЃњрЃЏрЃюрЃћрЃЉрЃљ

 ### рЃърЃарЃЮрЃфрЃћрЃАрЃў-рЃарЃЮрЃњрЃЮрЃарЃф
 рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ, рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃЏрЃБрЃерЃљрЃЋрЃЊрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃБрЃЉрЃарЃљрЃџрЃЮ рЃбрЃћрЃЦрЃАрЃбрЃў

 рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃў рЃљрЃарЃўрЃА HTML, рЃўрЃА рЃЊрЃљрЃќрЃўрЃљрЃюрЃЊрЃћрЃЉрЃљ, рЃЌрЃБ рЃљрЃа рЃњрЃљрЃЊрЃљрЃџрЃљрЃ«рЃљрЃЋрЃЌ `-p html` / `--process-as html` рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃА

 ### рЃцрЃўрЃџрЃбрЃарЃў
 рЃЌрЃљрЃЋрЃњрЃљрЃЊрЃљрЃАрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃДрЃЋрЃљрЃарЃБрЃџрЃЌрЃљрЃЌрЃЋрЃўрЃА: рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃљрЃА рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃўрЃљрЃарЃЮрЃЌ `-F` / `--filter` рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў.
 рЃњрЃљрЃАрЃљрЃцрЃўрЃџрЃбрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃАрЃљрЃюрЃљрЃЏ рЃЕрЃљрЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃцрЃљрЃўрЃџрЃБрЃа рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў

 рЃљрЃЏ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃњрЃќрЃљ JS рЃцрЃљрЃўрЃџрЃўрЃАрЃЎрЃћрЃю, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃА рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ `filter`

 `filter` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА `async` , рЃарЃљрЃЊрЃњрЃљрЃю рЃЏрЃљрЃАрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћрЃЉрЃљ `await`

 рЃАрЃљрЃюрЃљрЃЏ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃЊрЃўрЃАрЃЎрЃќрЃћ рЃЕрЃљрЃўрЃгрЃћрЃарЃћрЃЉрЃљ, рЃЏрЃЌрЃћрЃџрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў рЃњрЃљрЃЊрЃљрЃћрЃфрЃћрЃЏрЃљ `filter` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА, рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў

 рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ `filter` рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃЊрЃљрЃю рЃљрЃарЃўрЃА рЃўрЃА, рЃарЃљрЃф рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ рЃЕрЃљрЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃАрЃљрЃфрЃљрЃЋрЃерЃў
