 ## рІхрїІрЇЇ ріЦріЊ рІерїѕріЋрІўрЅЦ рІхрїІрЇЇ
 рЇЋрѕ«рЇїрѕйріЊрѕЇ ріГрЇЇрЅх рѕЮріЋрїГ рѕХрЇЇрЅхрІїрѕГ рїѕріЋрЅб рѕѕрѕўрѕєріЋ ріЦрІерѕъріерѕГріЕ ріљрІЇрЇб рІЇрѕхрїЦ ріЦрІерѕарѕФрѕЂ ріљрЅарѕГ
 рІерѕХрЇЇрЅхрІїрѕГ рібріЋрІ▒рѕхрЅхрѕфрІЇ рѕѕрЅЦрІЎ рІЊрѕўрЅ│рЅх рѕхрігрЅ│рѕЏ ріЕрЅБріЋрІФрІјрЅйріЋ рїђрѕЮрѕг рѕѕрѕЋрІЮрЅЦ ріЕрЅБріЋрІФрІјрЅй рѕИрїФрѕѕрѕЂрЇб
 рЅарЅЁрѕГрЅА рѕЦрѕФ ріарїБрѕЂрЇБ ріЦріЊ рѕїрѕІ рІерЅ░рѕарѕѕрЇЇріЕрЅх рѕЦрѕФ рІерѕѕріЮрѕЮ

 рѕхрѕѕрІџрѕЁ рїарЅЃрѕџ рѕХрЇЇрЅхрІїрѕГ рѕѕрѕўрї╗рЇЇ ріЦрѕъріГрѕФрѕѕрѕЂ ріЦріЊ рІФ ріЦріЋрІ░рѕџрѕ░рѕФ рѕѕрѕЏрІерЅх ріЦрѕъріГрѕФрѕѕрѕЂрЇб

 рЅхріЋрѕ╣ріЋ (рЅарЇЊрЅхрѕфрІ«ріЋ рЅаріЕрѕЇ рІѕрѕГрѕЃрІі рѕўрІІрї«)(https://www.patreon.com/cobbzilla) ріЦріЋрі│ріЋ ріерѕЇрЅЦ ріарІ░ріЋрЅЃрѕѕрѕЂрЇб

 ## рѕўрїФріЋ
 рІерЅхріЦрІЏрІЮ рѕўрѕхрѕўрѕГ рѕўрѕ│рѕфрІФрІЇріЋ рѕѕрѕўрїарЅђрѕЮ `npm` рІѕрІГрѕЮ `yarn` ріЋ рЅарѕўрїарЅђрѕЮ рІГрїФріЉрЇА-

    npm install -g hokeylization
    yarn global add hokeylization

 ріЦріЋрІ░ рЅцрЅ░-рѕўрїйрѕљрЇЇрЅх рѕѕрѕўрїарЅђрѕЮ `lite` рїФріЋрЇБ рІГрѕЁрѕЮ рЅарїБрѕЮ рІФріљрѕ░ ріљрІЇ

    npm install -g hokeylization-lite
    yarn global add hokeylization-lite

 ріерІџрІФ рѕѕ `hokey` рЅхрІЋрІЏрІЮ ріЦрїѕрІЏріЋ рІГрѕўрѕЇріерЅ▒рЇА-

    hokey --help
    hokey -h

 рІЇрЇЁрІЊрЅх рЅаріЦрѕГрѕхрІј рЅІріЋрЅІ рІѕрІГрѕЮ рЅарѕїрѕІ рЅІріЋрЅІ рѕЏрІерЅх рІГрЇѕрѕЇрїІрѕЅ?

 `hokey` ріаріФрЅБрЅб рЅ░рѕѕрІІрІІрї«рЅй рЅарѕФрѕх-рѕ░рѕГ рѕѕрѕЏрїЇріўрЅх рІГрѕъріГрѕФрѕЇрЇб

 `LC_ALL` ріаріФрЅБрЅб рЅ░рѕѕрІІрІІрїГріЋ рЅарѕЏрЅђріЊрЅарѕГ рЅІріЋрЅІріЋ рѕЏрѕхрїѕрІ░рІх рЅхрЅйрѕІрѕѕрѕЁрЇА-

    LC_ALL=it hokey --help

 ріарѕхрЅ░рІЇрѕЇ `hokeylization-lite` рІерЅхріЦрІЏрІЮ ріЦрїѕрІЏ рІерѕџрїѕріўрІЇ рЅаріЦріЋрїЇрѕірІЮріЏ рЅЦрЅ╗ ріљрІЇрЇб

 ## ріарІўрїѕрїЃрїђрЅх
 рІеріЦрѕГрѕхрІјріЋ рІеGoogle рЅхрѕГрїЅрѕЮ рЇЋрѕ«рїђріГрЅх рѕѕрѕўрѕѕрІерЅх рІе `GOOGLE_TRANSLATE_PROJECT_ID` Рђ║ ріаріФрЅБрЅбріЋ рЅ░рѕѕрІІрІІрїГ рІФрІўрїІрїЂ

 рІе `GOOGLE_APPLICATION_CREDENTIALS` Рђ║ ріаріФрЅБрЅбріЋ рЅ░рѕѕрІІрІІрїГ рІѕрІ░ рІѕрѕерІирЅИрІЇ рІеJSON рѕЮрѕхріГрѕГріљрЅХрЅй рІФрЅђріЊрЅЦрѕЕ
 рЅа Google рІ░рѕўріЊ рѕІрІГ рѕЏрѕерїІрїѕрїФ ріЦріЋрІ┤рЅх ріЦріЋрІ░рѕџрѕ░рѕФ ріФрІѕрЅЂ рЅаріІрѕІ (ріарѕхрІ░рѕ│рЅй рѕірѕєріЋ рІГрЅйрѕІрѕЇ)

 ріерѕЮріЋрїЕ рі«рІх ріЦрІерѕ«рїА ріерѕєріљрЇБ ріЦріљрІџрѕЁріЋ рЅарѕЮріЋрїЕ рІЇрѕхрїЦ `.env` рЇІрІГрѕЇ рІЇрѕхрїЦ рѕЏрѕхрїѕрЅБрЅхрѕЮ рІГрЅйрѕІрѕЅрЇб
 рѕЏрІЇрїФ рЅа [dotenv](https://www.npmjs.com/package/dotenv) рЅаріЕрѕЇ рЅарѕѓрІ░рЅх рѕІрІГ рІГрїФріЊрѕЅ

 ## рІерїЃрЅФрѕхріГрѕфрЇЋрЅх рѕЋрЅЦрѕерЅЂрѕЮрЇі рѕўрѕГрїЃ рЇІрІГрѕЇріЋ рѕўрЅ░рѕГрїјрѕЮ
 рІерѕЋрЅЦрѕерЅЂрѕЮрЇі рѕ░ріЋрїарѕерІЦрІј **ріерІџрѕЁ рѕЂрѕѕрЅх рЅЁрїЙрЅй рІЇрѕхрїЦ рЅаріаріЋрІ▒ рЅарїЃрЅФрѕхріГрѕфрЇЋрЅх рЇІрІГрѕЇ рІЇрѕхрїЦ рѕўрѕєріЋ ріарѕѕрЅарЅхрЇА-

 ES6 рІѕрІ░ рІЇрїГ рѕўрѕІріГрЇА

    export default {
      string_key: "some value",
      another_key: "another value",
      ... more keys ...
    }

 CommonJS рІѕрІ░ рІЇрїГ рѕўрѕІріГ

    module.exports = {
      string_key: "some value",
      another_key: "another value",
      ... more keys ...
    }

 рІГрѕЁ рЇІрІГрѕЇ `myfile.en.js` рЅ░рЅЦрѕј ріерЅ░рѕ░рІерѕў рІѕрІ░ рѕхрЇЊріњрѕй ріЦріЊ рїђрѕГрѕўріЋ рЅарѕџріерЅ░рѕѕрІЇ рѕўрЅ░рѕГрїјрѕЮ рЅхрЅйрѕІрѕѕрѕЁрЇА-

    hokey -l es,de -o myfile.LANG.js myfile.en.js

 ріерѕІрІГ рІФрѕѕрІЇ `LANG` рѕЇрІЕ ріљрІЇ - рЅарІџрѕЁ рѕўрѕ│рѕфрІФ рІЇрѕхрїЦ рІерЅ░рІФрІў рЅЃрѕЇ ріљрІЇ!

 `LANG` рЇІрІГрѕјрЅй рІерЅІріЋрЅІ рі«рІх рЅ░рЅ░ріГрЅирѕЇрЇб

 рѕхрѕѕрІџрѕЁ ріерѕІрІГ рІФрѕѕрІЇ рЅхрІЋрІЏрІЮ рЇІрІГрѕјрЅйріЋ рІГрЇѕрїЦрѕФрѕЇ:

    myfile.es.js
    myfile.de.js

 `-l` / `--languages` ріарѕЏрѕФрїГ рЅаріљрїарѕІ рѕ░рѕерІЮ рІерЅ░рѕѕрІЕ рІе ISO рЅІріЋрЅІ рі«рІХрЅй рІЮрѕГрІЮрѕГ ріљрІЇрЇб
 [рЅаGoogle рЅхрѕГрїЅрѕЮ рІерЅ░рІ░рїѕрЇѕ](https://cloud.google.com/translate/docs/languages)

 рІерІЇрїцрЅх рЇІрІГрѕЅ рЅђрІхрѕърІЇріЉ ріФрѕѕ, рІерЅхріърЅ╣ рЅЂрѕЇрЇјрЅй ріЦріЋрІ│рѕЅ рѕѕрѕЏрІѕрЅЁ рІГрѕўрѕерѕўрѕФрѕЇ.
 ріљрЅБрѕГ рЅЂрѕЇрЇјрЅй ріарІГрЅ░рѕерїјрѕЎрѕЮрЇб рІерїјрІ░рѕЅ рЅЂрѕЇрЇјрЅй рЅхрѕГрїЅрѕърЅй рІГрЇѕрїарѕФрѕЅ ріЦріЊ рІГрЅ│ріерѕІрѕЅрЇб
 ріЦрѕхріе JS ріљрїѕрѕГ рѕўрїерѕерѕ╗ рІхрѕерѕх. рїарЅЁрѕІрѕІрІЇ рЇІрІГрѕЇ рѕЂрѕЇ рїірІю ріЦріЋрІ░рїѕріЊ рІГрЇЃрЇІрѕЇрЇб

 рѕЂрѕЅріЋрѕЮ рЅЂрѕЇрЇјрЅй `-f` / `--force` ріарѕЏрѕФрїГріЋ рІГрїарЅђрѕЎ

 ## рІерїйрѕЉрЇЇ рЇІрІГрѕјрЅйріЋ рѕЏрІЇрїФ рѕўрЅ░рѕГрїјрѕЮ
 рІерЇІрІГрѕјрЅйріЋ рѕЏрІЇрїФ рѕўрЅ░рѕГрїјрѕЮрѕЮ рЅхрЅйрѕІрѕѕрѕЁрЇб рѕєріфрѕІрІГрІюрѕйріЋ ріЦрІФріЋрІ│ріЋрІ▒ріЋ рЅарЅ░рІ░рїІрїІрѕџ рІГрїјрЅаріЏрѕЇрЇб
 рЅарѕЏрІЇрїФрІЇ рІЇрѕхрїЦ рЇІрІГрѕЇ рІФрІхрѕГрїЅ ріЦріЊ рІГрІўрЅ▒ріЋ рЅа Google рЅхрѕГрїЅрѕЮ рЅаріЕрѕЇ рІФрѕѓрІ▒ ріЦріЊ рІЇрїцрЅ▒ріЋ рІФрѕхрЅђрѕЮрїА
 рЅарЅ░рѕѕрІе рІерѕЏрІЇрїФ рІЏрЇЇ рІЇрѕхрїЦ рЅарЅ░рѕўрѕ│рѕ│рІГ рѕўрѕЇріЕ рІѕрІ░рЅ░рѕ░рІерѕў рЇІрІГрѕЇ

 рІерЅхрѕГрїЅрѕЮрІј рІњрѕІрѕЏ рѕЏрІЇрїФ рѕ▓рѕєріЋ рІГрѕЁ рѕЂріљрЅ│ ріљрЅЁрЅирѕЇрЇб

 `-o` / `--outfile` ріарѕЏрѕФрїГ рІерІЇрїцрЅх рѕЏрІЇрїФрІЇріЋ рІГрїѕрѕЇрї╗рѕЇ

 **рЅхрѕЇрЅЁ рѕЏрѕхрїаріЋрЅђрЅѓрІФ**рЇА рѕЏрІЇрїФрІјрЅйріЋ рЅарѕџрЅ░рѕерїЅрѕЎрЅарЅх рїірІю ** ріарЅхрѕхрїА ** рІерІЇрїцрЅх рѕЏрІЇрїФ рІГрїЇрѕѕрї╣
 рІФ рЅарїЇрЅцрЅх рѕЏрІЇрїФрІј рІЇрѕхрїЦ ріљрІЇ! рІГрѕЁріЋ ріФрІ░рѕерїЇріГрЇА-
 * рѕЏрѕѕрЅѓрІФ рІерѕїрѕѕрІЇ рѕўрІ░рїІрїѕрѕЮ рѕЏріљрѕ│рѕ│рЅхрЇб
 * рІерїЅрїЇрѕЇ рѕѓрѕ│рЅЦрІјріЋ рІФрѕѓрІ▒
 * рІ▓рѕхріГрІјріЋ рІГрѕЎрѕЅ
 * рЅхріЋрѕй рѕўрІЮріЊріЊрЅх

 *рІерѕЏрІГрІ░рѕерїЇрЅарЅх* рѕЮрѕ│рѕї рІГріИрІЇрѕЇрѕЁрЇА-

    hokey -l es -o templates/es templates # <--- DON'T DO THIS!

 рІГрѕЁ рѕ▓рѕ░рѕФрЇБ рІерЅ░рЅ░рѕерїјрѕЎ рЇІрІГрѕјрЅй рІѕрІ░ `templates/es` рЇБ ріЦріЊ рЅарІџрѕЁрѕЮ ріарІ▓рѕх рІГрѕєріЊрѕЅ
 рІерѕџрЅ░рѕерїјрѕЎ рѕЮріЋрїГ рЇІрІГрѕјрЅйрЇБ `templates/` рѕхрѕГ рѕхрѕѕрѕєріЉ рІГрѕЁ рѕѓрІ░рЅх рІГрЅђрїЦрѕІрѕЇ
 рѕѕрІўрѕІрѕѕрѕЮ рЇБ ріарЅ│рІхрѕГрїЅ!

 ### рЅхріГріГрѕѕріЏ ріарїарЅЃрЅђрѕЮ
 ріЦрѕ║рЇБ рЅарѕЏрІЇрїФ рІЇрѕхрїЦ ріаріЋрІ│ріЋрІх рІерібрѕюрІГрѕЇ ріарЅЦріљрЅХрЅй ріарѕЅрѕЁ ріЦріЋрЅарѕЇрЇА-

 ріЦріљрІџрѕЁріЋ рѕЂрѕЅ рІѕрІ░ рѕхрЇЊріњрѕй ріЦріЊ рїђрѕГрѕўріЋріЏ рѕѕрѕўрЅ░рѕГрїјрѕЮрЇБ ріарѕѓрІхрЇА-

    hokey -l es,de -o templates/email/LANG templates/email/en

 ріерѕІрІГ рЅБрѕѕрІЇ рІЇрѕхрїЦ `LANG` рІерЅ░рІФрІў рЅЃрѕЇ ріљрІЇ ріЦріЊ рЅа ISO рЅІріЋрЅІ рі«рІх рІГрЅ░ріФрѕЇрЇб

 ріерѕІрІГ рѕ▓рѕ«рїЦ рѕЮріЋ рІГріерѕ░рЅ│рѕЇ
 * `templates/email/es` " ріЦріЊ `templates/email/de` рѕЏрІЇрїФрІјрЅй рІГрЇѕрїарѕФрѕЅ (ріерѕїрѕЅ)
 * `templates/email/en` рІЇрѕхрїЦ рІФрѕѕ ріЦрІФріЋрІ│ріЋрІ▒ рЇІрІГрѕЇ рІѕрІ░ рѕхрЇЊріњрѕй ріЦріЊ рїђрѕГрѕўріЋріЏ рІГрЅ░рѕерїјрѕЏрѕЇ
 `-f` / `--force` ріФрѕЇрЅ░рїарЅђрѕЮріГ рЅарЅђрѕГ ріљрЅБрѕГ рІерІЇрїцрЅх рЇІрІГрѕјрЅй ріЦріЋрІ░рїѕріЊ ріарІГрЇѕрїарѕЕрѕЮрЇб
 * ріаріЋрІх ріарІГріљрЅх рІерѕЏрІЇрїФ рѕўрІІрЅЁрѕГ ріЦріЊ рЇІрІГрѕјрЅй рЅа `es` ріЦріЊ `de` рІЇрѕхрїЦ рЅа'en' рѕхрѕГ `en`

 #рѕїрѕјрЅй ріарѕЏрѕФрї«рЅй

 ### рІ░рѕерЅЁ ріарѕѓрІх
 рѕЮріЋ ріЦріЋрІ░рѕџрІ░рѕерїЇ рѕѕрѕЏрѕ│рІерЅх `-n` / `--dry-run` ріЋ рІГрѕѕрЇЅрЇБ ріљрїѕрѕГ рїЇріЋ рѕЮріЋрѕЮ ріарІГріљрЅх рІеріцрЇњріарІГ рїЦрѕф ріарІФрІхрѕГрїЅ рІѕрІГрѕЮ рѕЮріЋрѕЮ ріарІГріљрЅх рЇІрІГрѕЇ ріарІГрї╗рЇЅ

 ### ріарѕхрїѕрІхрІ▒
 рЅхрѕГрїЅрѕърЅйріЋ рѕЂрѕЇрїірІю рѕѕрѕЏрІ░рѕх `-f` / `--force` ріЋ рІГрѕѕрЇЅрЇБ рѕЮріЋрѕЮ ріЦріЋрі│ріЋ рЅђрІ░рѕЮ рЅЦрѕѕрІЇ рЅбріќрѕЕрѕЮ

 ### рїЇрїЦрѕџрІФ
 рЅарѕЏрІЇрїФ-рѕърІх рІЇрѕхрїЦ рѕ▓рѕёрІ▒ рІерѕџріеріЊрІѕріЉрЅхріЋ рЇІрІГрѕјрЅй рѕѕрѕўрїѕрІ░рЅЦ `-m` / `--match` ріЋ рІГрѕѕрЇЅ

 рѕЂрѕЇрїірІю *ріЦрІФріЋрІ│ріЋрІ▒ріЋ* рЇІрІГрѕЇ рЅарѕЮріЋрїГ рѕЏрІЇрїФрІј рІЇрѕхрїЦ рІѕрІ░ рІњрѕІрѕЏ рѕЏрІЇрїФрІј рѕўрЅ░рѕГрїјрѕЮ рѕІрІГрЇѕрѕЇрїЅ рІГрЅйрѕІрѕЅрЇб

 `-m` / `--match` ріарѕЏрѕФрїГ рІІрїІ regex ріљрІЇ (рІерѕ╝рѕЇ рїЦрЅЁрѕх рІ░ріЋрЅдрЅйріЋ рЅ░рїаріЋрЅђрЅЂ!) рІерѕџрїѕрѕЇрїй
 рІерЅхріърЅ╣ рЇІрІГрѕјрЅй рѕўрЅ░рѕГрїјрѕЮ ріарѕѕрЅБрЅИрІЇ

 рЅарѕџрїарѕФрїарѕЕрЅарЅх рїірІюрЇБ рІерЅхріЏрІјрЅ╣ рЇІрІГрѕјрЅй ріЦріЋрІ░рѕџрЅ░рѕерїјрѕЎ рѕѕрѕЏрІерЅх рІГрѕЁріЋ ріарѕЏрѕФрїГ `-n` / `--dry-run` рїІрѕГ рѕЏрїБрѕўрѕГ рІГрЅйрѕІрѕЅрЇб

 ### ріарІФріФрЅхрЅхрѕЮрЇб
 ріаріЋрІ│ріЋрІх рїірІю рІеріЦрѕГрѕхрІј `-m` ріерЅЦрІЎ рЇІрІГрѕјрЅй рїІрѕГ рІГрІЏрѕўрІ│рѕЇрЇб рЅарїЇрѕЇрЇЁ рѕѕрѕЏрїЇрѕѕрѕЇ `-e` / `--excludes` ріарѕЏрѕФрїГ рІГрїарЅђрѕЎ
 ріарѕѕрЅарѕѕрІџрІФ рѕірІЏрѕўрІ▒ рІерѕџрЅйрѕЅ рЇІрІГрѕјрЅй

 рЅарЅдрЅ│ рЅ░рѕѕрІФрІГрЅ░рІЇ рЅарѕГріФрЅ│ regexes рѕўрІўрѕГрІўрѕГ рІГрЅйрѕІрѕЅрЇб

 рІерЅ░рѕѕрѕўрІ░рІЇ ріарїарЅЃрЅђрѕЮ `--excludes node_modules dist \.git build tmp`

 ### рІеріЦрїЁ рѕўрІФрІБрІјрЅй
 рІерѕџрЅ░рѕерїјрѕЎрЅх рѕЋрЅЦрѕерЅЂрѕЮрЇірІјрЅй `{{ handlebars }}` ріарЅЦріљрЅХрЅйріЋ рѕірІГрІЎ рІГрЅйрѕІрѕЅрЇБ рІѕрІГ рѕЂрѕѕрЅх рІѕрІГрѕЮ рѕХрѕхрЅх рїЦрѕЮрІЮ рЅЁріЋрЇЇ рІФрѕІрЅИрІЇрЇб

 рЅаріЦріљрІџрІФ ріарЅЦріљрЅХрЅй рІЇрѕхрїЦ рІФрѕЅрЅх ріљрїѕрѕ«рЅй ріЦріЋрІ▓рЅ░рѕерїјрѕЎ *ріарЅхрЇѕрѕЇрїЅрѕЮ*

 `-H` / `--handlebars` рЅБріЋрІ▓рѕФ ріЦрѕѕрЇЇ ріЦріЊ рЅа` `{{ ... }}` рІЇрѕхрїЦ рІФрѕѕ рѕЏріЋріЏрІЇрѕЮ ріљрїѕрѕГ ріарІГрЅ░рѕерїјрѕЮрѕЮ

 ### рѕЮрѕЇріГрЅх рѕЏрІхрѕерїірІФ
 Markdown рїйрѕЉрЇЇрѕЮ рѕєріљ ріцрЅйрЅ▓ріцрѕЮріцрѕЇ ріарІГрІ░рѕѕрѕЮрЇБ рѕхрѕѕрІџрѕЁ рїјрїЇрѕЇ рЅ░рѕГрїЊрѕџ ріаріЋрІ│ріЋрІх рЅйрїЇрѕ«рЅй ріарѕЅрЅх

 `-M` / `--markdown` рЅБріЋрІ▓рѕФ рѕѕрѕЏрѕГріГ рЇІрІГрѕјрЅй рѕЇрІЕ ріарІФрІФрІЮріЋ рІФрѕхрЅйрѕІрѕЇ

 рЅарѕЏрѕГріГ рѕЏрІхрѕерїірІФ рЇІрІГрѕјрЅйрЇБ `-M` рЅБріЋрІ▓рѕФ ріФрѕЇрЅ░рїарЅђрѕЎрЇБ рѕЮріЊрѕЇрЅБрЅх ріЦріљрІџрѕЁріЋ рЅйрїЇрѕ«рЅй рѕірІФрїѕріЎ рІГрЅйрѕІрѕЅрЇА-
 * рІерЅ░рЅарѕІрѕ╣ ріарїѕріЊріърЅйрЇб рЅарЅхрѕГрїЅрѕЎ рІЇрѕхрїЦрЇБ рІерѕЏрѕГріГ рѕЏрІхрѕерїірІФ ріарїѕріЊріЮ рѕўрїЇрѕѕрїФ ріФрѕѕрЅђ рЅаріІрѕІ ( `]` рїІрѕГ) рїЇріЋ рІерѕЋрІІ рЅЂрѕЮрЇі рІГрЅ│рІФрѕЇ
 рІерІњрѕІрѕЏрІЇ рѕЏрїѕріЊріЏ ріерѕўрїђрѕўрѕЕ рЅарЇірЅх ( `(` рїІрѕГ)рЇб рІГрѕЁ рѕЮрѕЇріГрЅх рѕЏрІхрѕерїірІФрІЇ рЅарѕхрѕЁрЅ░рЅх ріЦріЋрІ▓рѕ░рѕФ рІФрІ░рѕГрїѕрІІрѕЇрЇБ ріЦріЊ ріарїѕріЊріЎ
 рѕ░ріљрІ▒ріЋ рѕ▓рѕўрѕѕріерЅ▒ рЅ░рѕ░рЅЦрѕ»рѕЇ.
 * рі«рІх рЅЦрѕјрі«рЅй рЅ░рЅ░рѕГрїЅрѕўрІІрѕЇрЇб рїјрїЇрѕЇ рЅ░рѕГрїЊрѕџ рѕЏрѕГріГ рІ│рІЇріЋ рѕЮріЋ рі«рІх ріЦріЋрІ░рѕџрІФрѕхрЅЦ ріЦріЊ рѕЮріЋ ріЦріЋрІ░рѕЏрІФрІЇрЅЁ ріарІФрІЇрЅЁрѕЮ
 * рѕѕрЅ░рїарѕѕрЇЅ рІері«рІх рЅЦрѕјрі«рЅй рЅхріГріГрѕѕріЏ рІФрѕЇрѕєріљ ріГрЇЇрЅ░рЅхрЇб ріГрЇЇрЅ░рЅх рЅарЅхрѕГрїЅрѕЮ рІЇрѕхрїЦ рѕѕрѕЏрЅєрІерЅх ріарѕхрЅИрїІрѕф ріљрІЇ
 * рѕЂрѕЇрїірІю рѕЏрѕѕрЅх рІГрЅ╗рѕІрѕЇ ріЦрІЇріљрЅ░ріЏ ріЦрѕ┤рЅХрЅй ріЦріЋрІ▓рѕєріЉ рЅарѕџрЇѕрѕЇрїЅрЅарЅх рїірІю рЅа `backticks` Рђ║ рІЇрѕхрїЦ рІФрѕЅ ріљрїѕрѕ«рЅй рІГрЅ░рѕерїјрѕЏрѕЅ

 `-M` / `--markdown` рЅБріЋрІ▓рѕФ рѕ▓ріљрЅЃрЇА-
 * рѕЦрѕГрІЊрЅ░ рїЦрѕѕрЅх `](` рІѕрІ░ `](` ) рІГрѕ░рЅарѕ░рЅБрѕЇ рѕхрѕѕрІџрѕЁ рІерЅ░рѕ░рЅарѕе рѕЮрѕЇріГрЅх рѕЏрІхрѕерїірІФ ріарїѕріЊріърЅйріЋ рІФрѕхрЅ░ріФріГрѕІрѕЇ
 * "рЅхрѕГрїЅрѕЮ рІерѕѕрѕЮ" рѕўрїарЅЁрѕѕрІФ рЅарЅ░рїарѕѕрЇЅ рІері«рІх рЅЦрѕјрі«рЅй рІЎрѕфрІФ рІГрІ░рѕерїІрѕЇрЇБ рЅ░рїѕрЅбрІЇріЋ рІЇрѕхрїа-рїЇріЋрІЏрЅц рЅарѕўрїарЅарЅЁ ріЦріЊ ріарѕѕрѕўрЅ░рѕерїјрѕЏрЅИрІЇріЋ рІФрѕерїІрїЇрїБрѕЇрЇб
 * `backticks` рѕѕрѕЏрѕерїІрїѕрїЦ "рІерѕЏрІГрЅ░рѕерїјрѕЮ" рѕўрїарЅЁрѕѕрІФ рЅарЇЁрѕЂрЇЇ рІЎрѕфрІФ рЅаРђ╣backticksРђ║ рІЇрѕхрїЦ рІГрЅђрѕўрїБрѕЇ

 ### рѕѓрІ░рЅх-ріЦріЋрІ░
 рЅарЅ░рѕѕрѕЮрІХ рѕЂрѕЅрѕЮ ріљрїѕрѕГ ріЦріЋрІ░ рїЇрѕЇрїй рїйрѕЉрЇЇ ріљрІЇ рІерѕџрѕ░рѕФрІЇрЇб

 рІГрІўрЅхрѕЁ HTML ріерѕєріљ `-p html` / `--process-as html` ріарѕЏрѕФрїГріЋ ріФрѕІрѕѕрЇЇріГ рЅарѕхрЅ░рЅђрѕГ рІГрЅарѕІрѕ╗рѕЇрЇб

 ### ріарїБрѕФ
 рѕѕрїђрЅЦрІ░ріЏрІјрЅ╣рЇА рЇІрІГрѕјрЅйріЋ рЅарѕЏрІЇрїФ рІЇрѕхрїЦ `-F` / `--filter` ріарѕЏрѕФрїГ рѕЏрѕѕрЇЇ рІГрЅйрѕІрѕЅрЇб
 рЅарЇІрІГрѕЇ рѕхрѕГрІЊрЅ▒ рѕІрІГ ріерѕўрї╗рЇЅ рЅарЇірЅх рІЇрїцрЅ▒ріЋ рѕѕрѕЏрїБрѕФрЅх

 рІерІџрѕЁ ріарѕЏрѕФрїГ рІІрїІ `filter` рІерѕџрЅБрѕЇ рЅ░рїЇрЅБрѕГ рІѕрІ░ рІЇрїГ рІѕрІ░ рІЇрїГ рІѕрІ░ рІЇрїГ рІѕрІ░ рІЇрїГ рІерѕџрѕІріГ рІерїёріцрѕх рЇІрІГрѕЇ рІ▒ріФ рѕўрѕєріЋ ріарѕѕрЅарЅхрЇб

 `filter` рЅ░рїЇрЅБрѕГ `async` ріарѕхрѕЮрѕГ' рѕўрѕєріЋ ріарѕѕрЅарЅх рѕЮріГріЋрІФрЅ▒рѕЮ `await` рЅаріЦрѕ▒ рѕІрІГ рІГрїарѕФрѕЇ

 рЇІрІГрѕјрЅй рІѕрІ░ рІ▓рѕхріГ `filter` рЅарЇірЅхрЇБ рІерЇІрІГрѕЅ рІГрІўрЅХрЅй рЅарѕЎрѕЅ рІѕрІ░ РђюрѕЏрїБрѕфрІФРђЮ рЅ░рїЇрЅБрѕГ ріЦріЋрІ░ рѕЋрЅЦрѕерЅЂрѕЮрЇі рІГрЅ░рѕІрѕѕрЇІрѕЅ

 ріе `filter` рЅ░рїЇрЅБрѕГ рІерѕџрїѕріўрІЇ рІерѕўрѕўрѕѕрѕ╗ ріЦрѕ┤рЅх рІѕрІ░ рѕЏріерѕЏрЅ╗ рІерѕџрї╗рЇѕрІЇ ріљрІЇрЇб

 рѕхрѕѕрІџрѕЁ рЅарѕўрїерѕерѕ╗ рЅарѕџрЇЃрЇѕрІЇ рѕІрІГ рѕЎрѕЅ рЅЂрїЦрїЦрѕГ ріарѕѕрѕЁ
